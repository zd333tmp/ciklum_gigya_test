Gigya Test Appp

Generated with Angular CLI version 1.0.2. Uses ngrx solutions for state management.

Development Env

Prepare environment:

  1. be sure to have node (7.6 or later), npm (4.1 or later) and Yarn (0.21 or later) installed
  2. npm install -g @angular/cli@1.X
  3. ng set --global packageManager=yarn
  4. yarn install

Use next command in dev mode:

  • ng serve for a dev server, then navigate to http://localhost:4200/
  • ng lint to lint sources
  • ng test to run tests

Using state dev tools (including time travel debugging ;) ):

  1. download and install the Redux Devtools Extension
  2. run app in dev mode and launch extension, then enjoy the magic

Build

  1. prepare environment (steps 1-3 Development Env section)
  2. rm -rf node_modules
  3. yarn install
  4. define API connection details in src/environments/environment.prod.ts
  5. ng build --prod
  6. pick-up result artifact at dist folder

Documentation Auto Generation

Be sure to have compodoc globally installed (npm install -g @compodoc/compodoc).

  1. rm -rf documentation
  2. mkdir documentation
  3. compodoc -p tsconfig.json
  4. pick-up result artifact at documentation folder
